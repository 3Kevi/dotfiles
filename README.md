### Welcome!
Here you can find dotfiles for my Linux system.

### Overview:
- OS: Void
- Window manager: bspwm
- Terminal: kitty
- Shell: zsh
- Bar: Polybar
- Compositor: picom
- Program launcher: Rofi
- PDF viewer: Zathura
- Hotkey daemon: sxhkd
