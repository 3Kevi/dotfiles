## Welcome!
Here you can find my Void Linux dotfiles.

## Overview
- OS: Void
- Window manager: bspwm
- Terminal: [tym](https://github.com/endaaman/tym)
- Shell: zsh
- Bar: Polybar
- Compositor: picom
- Program launcher: Rofi
- PDF viewer: Zathura
- Hotkey daemon: sxhkd
